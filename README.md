# Installation

Clone repository ✈️ 
```$ git clone https://github.com/dmitryglitch/react-router-redux-toolkit-sass-starter-kit.git ```
``` $ cd <my-project-name> ```

Install the project dependencies, npm install will suffice 🧶
```$ npm install ```

And run project 🚀
```$ npm run start ```
